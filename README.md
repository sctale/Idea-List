# 闪念 · Idea List

> 灵光一闪，随手记下。一个极简的灵感捕捉 App。

## 特性

- **零压力输入** — 打开即写，Enter 即存，不打断思路
- **即时保存** — 灵感自动存入本地，无需手动保存
- **小清新风格** — 薄荷绿 + 暖白，轻松自在无压力
- **三页架构** — 记录页 / 汇总页 / 关于页，清晰分工
- **智能搜索** — 汇总页支持关键词搜索，日期分组展示
- **丝滑动画** — 灵感卡片弹性动画、页面切换淡入淡出
- **纯本地** — 数据存在浏览器 LocalStorage，无需联网、无需注册

## 页面

| 页面 | 功能 |
|------|------|
| 记录 | 快速输入灵感，展示最近5条，时段问候语 |
| 汇总 | 全部灵感列表，搜索过滤，日期分组，今日统计 |
| 关于 | App 信息、特色功能、技术栈、GitHub 链接 |

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
