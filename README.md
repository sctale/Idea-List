# 闪念 · Idea List

> 灵光一闪，随手记下。一个极简的灵感捕捉 App。

## 特性

- **零压力输入** — 打开即写，Enter 即存，不打断思路
- **即时保存** — 灵感自动存入本地，无需手动保存
- **相对时间** — "刚刚"、"5 分钟前"，直觉感知时间流逝
- **丝滑动画** — 每条灵感以弹性动画出现，删除时滑出
- **深色主题** — 午夜蓝底色 + 琥珀色闪电，护眼又有氛围感
- **纯本地** — 数据存在浏览器 LocalStorage，无需联网、无需注册

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
