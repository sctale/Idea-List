# Changelog

All notable changes to this project will be documented in this file.

## [0.1.1] - 2026-06-17

### Changed
- 全面重构色系：从小清新薄荷绿 + 暖白替代深色主题
- 三页架构：记录页、汇总页、关于页，底部导航切换
- 记录页：增加时段问候语、字数统计、成功粒子动画、渐变空状态
- 汇总页：增加搜索功能、日期分组、今日统计徽章
- 关于页：App 信息卡片、特色功能列表、技术栈标签、GitHub 链接
- 底部导航：活跃态背景指示器、填充/描边图标切换
- 页面切换：轻柔淡入淡出动画
- Favicon 更新为小清新风格

## [0.1.0] - 2026-06-17

### Added
- 灵感快速输入：打开即写，Enter 发送，Shift+Enter 换行
- 灵感列表展示：卡片式布局，显示内容与相对时间
- 删除功能：hover 显示删除按钮，二次确认防误删
- 空状态引导：无灵感时展示呼吸动画引导语
- 本地持久化：LocalStorage 自动保存
- 深色主题：午夜蓝 + 琥珀色闪电视觉风格
- 弹性动画：灵感卡片出现/删除动画（Framer Motion）
- 输入框获焦发光效果
- 自定义滚动条
