# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-17

### Added
- 灵感快速输入：打开即写，Enter 发送，Shift+Enter 换行
- 灵感列表展示：卡片式布局，显示内容与相对时间
- 删除功能：hover 显示删除按钮，二次确认防误删
- 空状态引导：无灵感时展示呼吸动画引导语
- 本地持久化：LocalStorage 自动保存
- 深色主题：午夜蓝 + 琥珀色闪电视觉风格
- 弹性动画：灵感卡片出现/删除动画（Framer Motion）
- 输入框获焦发光效果
- 自定义滚动条
